# 通用格式初始化

## 1. 页面结构初始化

### 1.1 标题及网站图标初始化

``` html
    <!-- title 标签信息 -->
    <title>Anon v0.0.2</title>

    <!-- favicon 网站图标 -->
    <link rel="stylesheet" href="./assets/images/logo/favicon.ico" type="image/x-icon">
```

### 1.2 自定义 CSS 样式

```html
    <!-- custom css link 自定义 css 样式链接 -->
    <link rel="stylesheet" href="./assets/css/style.css">
```

### 1.3 自定义 Google 字体

``` HTML
    <!-- custom google font link 自定义谷歌字体链接 -->
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
    <link href="https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;500;600;700;800;900&display=swap" 
    rel="stylesheet">
```

### 1.4 自定义 JavaScript 脚本

``` HTML
    <!-- custom js link 自定义 JavaScript 链接 -->
    <script src="./assets/javascript/script.js"></script>
```

### 1.5 自定义字体图标

```html
    <!-- custom ionicon link 自定义字体图标链接 -->
    <script type="module" src="https://unpkg.com/ionicons@5.5.2/dist/ionicons/ionicons.esm.js"></script>
    <script nomodule src="https://unpkg.com/ionicons@5.5.2/dist/ionicons/ionicons.js"></script>
```
